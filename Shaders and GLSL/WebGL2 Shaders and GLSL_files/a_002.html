                          greggman Mod Muscle Code 3 years ago

                          Lately I've been using Visual Studio Code as my editor. I use the command line version of servez to serve pages locally for development.

                          As for your other question .vert, .frag, .glsl are file extensions people made up storing GLSL shader files. Some people put all shaders in files ending in .glsl, other people separate vertex shaders and fragment shaders by putting .vert or .frag. The advantage of putting shaders their own files that end in one of those extensions is it's easy to associate a syntax highlighter / linter

                          For example this plugin highlights / lints GLSL and it appears to look file at files that end in .vert, .frag, .glsl and a few others by default. I have not actually used it though.
